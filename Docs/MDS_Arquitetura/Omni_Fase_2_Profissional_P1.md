# Omni - Fase 2 Profissional (P1)

Prioridade da fase: `P1`  
Significado: obrigatorio para nivel profissional e release confiavel.

## Checklist da fase

- [ ] CI com build automatizado para as versoes de UE suportadas.
- [ ] Testes automatizados de runtime e de geracao do Forge.
- [ ] Versionamento de `Manifest` com migracao entre versoes.
- [ ] Logs estruturados e painel de debug (`OmniRecorder`).
- [ ] Metricas de performance com orcamento por sistema.
- [ ] Catalogo de erros com mensagens acionaveis no Forge.
- [ ] Politica de release com SemVer e changelog.
- [ ] Projeto de exemplo oficial com fluxo `Configure -> Generate -> Play`.
- [ ] Guia de upgrade entre versoes.
- [ ] Criterios de regressao antes de cada release.

## Gate da fase

- [ ] Pipeline CI verde.
- [ ] Testes de regressao verdes.
- [ ] Upgrade de versao validado sem quebrar projeto exemplo.
